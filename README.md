# Taipei-day-trip

## 功能
* 會員註冊系統
* 會員資料編輯（會員大頭貼存放AWS S3)
* 景點分類搜尋
* 首頁景點顯示Infinite scroll
* 行程預定
* 查看歷史訂單
* Tappay線上付款

## 前端
* HTML
* CSS
* Javascript

## 後端
* Python Flask

## 資料庫
* Mysql

## AWS雲端服務
* EC2
* S3

## 版本控制

* Github
